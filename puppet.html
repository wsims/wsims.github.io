Puppet Research Projects

Installer

The Challenge

Improve the usability of the Puppet Enterprise installer by conducting usability test sessions with internal and external participants.

Goals

  • Assess how well users are able to install in text-mode and recover from failed installations to meet product initiatives.
  • Find usability issues and areas for improvement to influence the future direction of the installer.

Usability Issues

To successfully install Puppet Enterprise, the user needs to set a console admin in the installer configuration file. Our findings showed that users were unaware they needed to set a password and the error message thrown was ineffective for troubleshooting the issue.

 

This this the error message that is displayed when there is a missing console admin password in the configuration file. Users interpreted the message as an issue with curly braces on line 33.

When participants reviewed the configuration file, they would ignore the “console_admin_password” field to focus on the formatting of line 33. Participants were unable to troubleshoot this error message unless prompted that it is unrelated to formatting.

Solutions

My findings resulted in the error message being redesigned to improve clarity when troubleshooting a missing console admin passoword.

We were able to test and validate this solution during the subsequent external usability test sessions with the installer. Users were selected based on the type of installer they were using. We chose two Puppet open source users who have never used the official installer and three who were actively using Puppet Enterprise. The improved error messaging caused participants to recover quickly and without prompting

However, the error still needed to be prevented in the first place. Based off my recommendations, a new design was implemented for increased visibility of the console admin password field that was validated during the benchmarking sessions. Overall, the goals for the study were met and users were successful at installing Puppet Enterprise.
