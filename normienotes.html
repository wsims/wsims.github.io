Normie Notes

The Challenge

Create a web application with computer science students at Oregon State University.

My Role

  • Design
  • Usability
  • Front-end development

Sketch Prototype

I first designed the concept for Normie Notes in sketch before moving on to front-end development. I intially thought that it was neccesary to have sign-in and account creation pages until I realized that it would be more efficient for students to login with their Oregon State accounts.

The first iteration of the homepage involved a table of notes that are sorted by some sort of arbitrary rating system. I first thought that we would use global search to find notes, but eventually decided that a dropdown facited search be superior. This is because a dropdown search eliminates the possibility of using the incorrect course number or mispelling a professor's name.
